ALPS/looper

The ALPS/looper package is one of the applications of the ALPS project. It provides multi-cluster quantum Monte Carlo algorithms for generic spin systems in the path-integral or SSE representations.

The applications supports the ferromagnetic and non-frustrated antiferromagnetic XXZ models on arbitrary lattices. Since it supports so-called freezing graphs, models with easy-axis (Ising-like) anisotropy can also be simulated very efficiently.

Compilation

./configure
make
In case that the ALPS library is installed under a directory other than the default one ($HOME/ALPS), please specify its location to the configure script by --prefix option.

Input parameters

In addition to the general input parameters of the ALPS scheduler library the spinmc application takes the following input parameters:

NameDefaultDescription
LATTICE_LIBRARYlattices.xmlpath to a file containing lattice descriptions
LATTICEname of the lattice
MODEL_LIBRARYmodels.xmlpath to a file containing model descriptions
MODELname of the model
representationpath integralthe type of representatioin, either "path integral" or "SSE"
temperaturethe temperature
MCSthe total Monte Carlo steps (including steps for thermalization)
thermalizationthe Monte Carlo steps for thermalization

In addition, the lattice (model) description can require further parameters (e.g. L or W) as specified in the lattice (model) description file.

Measurements

The following observables are measured by the spinmc application:

NameDescription
Energythe total energy of the system
Magnetizationthe z-component of the magnetization
|Magnetization|absolute value of the z-component of the magnetization
Magnetization^2square of the z-component of the magnetization
Staggered Magnetizationthe z-component of the staggered magnetization (only on bipartite lattices)
Staggered Magnetization^2square of the z-component of the staggered magnetization (only on bipartite lattices)
Susceptibilitythe uniform susceptibility
Cluster sizethe mean cluster size as a fraction of the lattice volume (only for cluster updates)
